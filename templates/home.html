Gesture Brightness Control

Welcome to the Gesture Brightness Control web application!

This project uses MediaPipe in the browser to track your hand gestures in real-time through your webcam.

✨ Key Features

  • Touchless Control: Adjust your screen brightness effortlessly without using a keyboard or mouse.
  • Real-time AI Tracking: Powered by Google's MediaPipe for highly accurate and fast hand landmark detection.
  • Visual Feedback: Features a dynamic on-screen bar that changes color (from red to green) based on the current brightness level.
  • Smooth Transitions: Implements a smoothing algorithm to prevent jittering caused by natural hand shakes.

🛠️ How it works

By pinching your index finger and thumb, the application calculates the distance between them. This distance is smoothly mapped to a visual brightness percentage in the live preview.

⚠️ Important Note

Please ensure you grant your browser permission to access your webcam. The application requires an active camera feed to detect your hand gestures.
